# EventMate

A scalable role-based event management platform designed to handle
event creation, seat booking, and waitlist management.

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- JWT Authentication

## Features
- Role-based access (Admin / Organizer / Attendee)
- Event creation and management
- RSVP with seat limits and waitlist
- Secure authentication

(Project under active development)
